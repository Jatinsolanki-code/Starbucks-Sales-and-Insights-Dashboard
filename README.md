# Starbucks-Sales-and-Insights-Dashboard

## 1. Project Title

☕ Starbucks Brewing Insights: Interactive Beverage Analytics Dashboard

A modern and interactive Power BI dashboard designed to analyze Starbucks beverage data through nutritional insights, beverage segmentation, caffeine trends, and category-level performance—helping transform raw beverage data into meaningful business intelligence.

## 2. Short Description

The Starbucks Brewing Insights Dashboard is an interactive Power BI report developed to explore beverage-related insights across Starbucks product categories. The dashboard focuses on analyzing nutritional indicators such as calories, sugar, caffeine, and beverage preparation styles to identify consumption trends and product characteristics.

This dashboard is intended for business analysts, data enthusiasts, marketing teams, and decision-makers who want to understand beverage performance and nutritional distribution through visual analytics.

## 3. Tech Stack

The dashboard was built using the following tools and technologies:

• 📊 Power BI Desktop – Main platform used to design and develop the dashboard visuals and interactions.
• 📂 Power Query – Used for data cleaning, transformation, and preparation before analysis.
• 🧠 DAX (Data Analysis Expressions) – Used to create measures, KPI calculations, filtering logic, and ranking.
• 📝 Data Modeling – Applied to structure relationships and improve dashboard performance.
• 🎨 Dashboard Design – Custom visual styling using Starbucks-inspired color themes and layout design principles.
• 📁 File Format – .pbix for development and .png for dashboard presentation.

## 4. Data Source

Source: Starbucks Beverage Nutrition Dataset

The dataset contains beverage-level information including:

• Beverage categories
• Beverage preparation types
• Calories
• Sugar values
• Caffeine content
• Nutritional attributes

The data was transformed and structured to support KPI analysis, filtering, and visual storytelling.

## 5. Features / Highlights

### • Business Problem

Modern beverage businesses generate extensive nutritional and product data, but extracting useful insights manually becomes difficult.

Questions such as:

• Which beverages contain the highest caffeine?
• Which categories contribute the highest calories?
• How is caffeine distributed across beverage types?
• What preparation styles are most common?

…require analytical visualization for quick understanding.

### • Goal of the Dashboard

To deliver an interactive beverage intelligence dashboard that:

• Enables exploration of Starbucks beverage data.
• Highlights nutritional trends across categories.
• Supports business decisions using KPI-driven analysis.
• Presents complex data in an easy-to-understand visual format.
